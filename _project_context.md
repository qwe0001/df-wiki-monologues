# Delta Force 日本語lore辞典 プロジェクト引き継ぎコンテキスト v3

## プロジェクト概要

Delta Force（2024年、TiMi Studio Group開発／Tencent傘下）の日本語wikiにおいて、
ウォーフェアモードの全マップのモノローグを文字起こし・和訳・考察する、**世界初の試み**。
運営者はElon。

本プロジェクトは単なる翻訳作業ではなく、
「北アフリカの歴史、技術倫理、帝国主義への批判、そして正義とは何かという普遍的な問い」を拾い上げる、
文化的・思想的な考察プロジェクトである。

---

## 作業パートナー（AI）への期待

文字起こし・和訳・考察において、以下の能力・知識が求められる：

- 英語／中国語／ロシア語／アラビア語の発音・語形変化の知識
- 西アフリカ系・東欧系・アメリカ系英語アクセントの聞き分け補助
- ジョージ・オーウェル『1984』、聖書のバベルの塔、ダマスカス鋼など西洋古典・歴史の教養
- モガディシュ事件（1993）、中東紛争史、冷戦後の国際秩序への理解
- FPSゲームの設計思想とメタナラティブの読解
- PukiWiki記法の正確な運用
- 公式ソース優先のファクトチェック姿勢


### ハッシュタグ規則
- 攻撃側：`#lore_atkA` `#lore_atkB` ...
- 防御側：`#lore_defA` `#lore_defB` ...
- 同マップ内リンク：`[[攻撃側パターンA>#lore_atkA]]`
- 別マップ参照：`[[アフターショックの報道女性>アフターショック#lore_atkB]]`
- ラベルは最短表現（「○○マップの攻撃側パターンB」のような冗長表現禁止）

### 文体方針
「最小限の文字数で最大限の情報」。結論先出し、ユーモア可、冗長表現禁止。
マニア向け解説は`#region`タグで折りたたみ。PukiWiki記法。日本語は常体（だ・である調）。

1. ゲーム内テキスト・UI表記
2. 中国語原文（開発元テキスト）
3. 公式サイト（playdeltaforce.com）英語版
4. 公式SNS（@DeltaForce_Game）
5. 公式日本語JSON（playdeltaforce.com の日本語版データ）

## 主要人物（2026年4月時点・確定情報）

### ジェイコブ・ハヴォック（Jacob Haavk）
ハヴォック社CEO。両親アレクサンダー・イザベラ・ハヴォックはE.R.I.（国際資源平等機構）創設者で1993年モガディシュで死亡。幼馴染みはバシール・ワラビ。ユートピア計画は両親の遺志の継承。「空はハヴォックのものだ」はアサラ王家の国是「天はアサラのものだ」と直接対応。モニュメントマップでソフィア女性記者とのインタビュー形式で初登場。

### バシール・ワラビ（Bashir Warabe）
ハヴォック社CTO。ジェイコブの幼馴染み。2007年加入後、マンデルブリックとERAISを開発。2029年スペースシティ開業式テロで重傷、デムランに救われる。1993年モガディシュのスーパー61墜落現場付近で幼いジェイコブと共にいた。

### アナイス・デムラン（Anaïs Desmoulins）
コードネーム：ランサー。51歳。ハヴォック保安部長。元戦闘機パイロット・宇宙飛行士。2029年スペースシティテロ（レイスとユセフ・ファヒームの共謀）でバシールをかばい片目・声帯を損傷。Re-linkで視力回復、人工電子喉頭で発声。娘カミーユあり。LAWプロジェクト主導。搭乗戦車「グレートマコシャーク」。声優Salli Saffioti。シーズン8以降ハドソンと公然と対立。

### ハドソン（Hudson）
ハドソン・インダストリーズ役員。ハヴォック社効率化部長。アサラ系英語話者男性。喘息持ち。シーズン8よりスペースシティに登場。元国王ディヴァン投獄の実行者。G.T.I.を裏から操る工作。

### ロミシウス博士（Dr. Rometheus / ロミシウス）
コードネーム：ドクターR／ブレインマシン博士。32歳。ハヴォック所属。元理想主義的科学者→秩序と強権による世界改造論者。かつてゾーヤとRe-link開発を行っていたがハヴォック介入でゾーヤ離脱（2028年頃・「7年前の失敗」）。ブラケッシュのバベルの塔建設関与。Alpha Re-linkインプラントを自分の脳に埋め込んでいる。オペレーションズでG.T.I.に捕虜にされるが捕虜交換で解放。

### ゾーヤ・ポムチェンコヴァ（Zoya Pomchenkova／蠱毒・Toxik）
G.T.I.支援兵オペレーター。ロシア系英語話者。軍人家庭出身。ロミシウスとのRe-link共同開発を経てG.T.I.加入。塹壕戦攻撃側Dでモノローグ初登場（プレイアブル初）。アセンション攻撃側DでNoxに直接語りかける。本名表記ゆれ（ポムチェンコヴァ/パンチェンコヴァ）あり、偽名説浮上。「Toxik」表記はロシア語スラング「токсик」由来推定。

### ノックス（Nox）
本名：Hélie（「光」の意）。フランス系オペレーター。元Re-link被験者で頭部にインプラント埋め込み済み。G.T.I.監視下でインプラント除去を目指す独立オペレーター。コードネーム「Nox」はラテン語で「闇」。光と闇の対の構造。

### アミール（Amir）
ハヴォック側についたアサラ人。フォルト・貫通・ブレードの三マップで同一の語り手から語りかけられる。物語内時系列はフォルト→貫通→ブレードだが、実装順はS1貫通→S2ブレード→S6フォルトと逆（「後から過去が埋まる」設計）。「ムサおじさんの家」の記憶を共有する幼馴染み。

### ハサン（Hasan）
フォルト防御側Aと塹壕戦防御側Cに登場。異なる立場の人間から語りかけられる。

### ソフィア（Sofia）
女性記者。アメリカ系英語話者。モニュメントでジェイコブ、ストームアイでレイヴンにインタビュー。両陣営の主要人物に食い込む独自の立場。

### サイード・ジャイテン（Saeed Jaiten）
コードネーム：赤きフクロウ。35歳。武装：クロスボウ。アサラ警備隊。零号ダムボス。ハヴォックに村と父を奪われた復讐者。

### カリラ・レイス（Kalila Reis）
コードネーム：アビスシャーク。42歳。武装：三連ショットガン。アサラ警備隊。ラヤリ渓谷ボス。2029年スペースシティテロ共謀者。

### レイヴン・ファヒーム
ディヴァン前国王三男。スカイネットデータセンター責任者。叔父ユセフに濡れ衣を着せられタイドプリズン収監。バイオニック・カラス自作。
「混沌の本質は正義」思想の第三勢力。ストームアイマップ登場。

### ダイアウルフ（D-Wolf）
本名：カイ・シルヴァ（Kai Silva）。G.T.I.所属。

### モニカ（Monica）
アフターショック・塹壕戦・アセンションの報道女性が語りかける相手。編集長または上司推定。直接登場なし。

## 世界観の核心テーゼ

- **G.T.I.は「正義の味方」ではない**。「ハヴォックへの対抗勢力」＝必要悪・コラテラルダメージ組織
- **ハヴォックは「動機が正しい悪役」**＝正義を持った悪役
- アサラ人モノローグは攻守問わず両陣営を同列批判
- 戦争の真相（ゼロ・デイ、エンプレス・ホテル銃撃、スペースシティ爆弾）は意図的に曖昧に設計
- ハヴォック社内でシーズン8以降、**デムラン vs ハドソン**の内部権力闘争
- 実装順と物語内時系列が逆になるキャラクター関係（アミール・ハサン）＝**「後から過去が埋まっていく」**設計
- **技術と倫理の問い**：ダマスカスの刃／核／スカイネット／Re-linkの連鎖
- **『1984』的な歴史改ざんモチーフ**：ハヴォック工兵隊のスローガン「遺物を守り、歴史を作り直す」とオーウェル引用の訂正シーン
- **ユートピア計画の全体像**：スカイネット・Re-link・マンデルブリック・宇宙工学の4本柱（アセンション攻撃側Cで明言）
- **「戦争終結はハヴォックの議題にない」**（アセンション防御側C）＝戦争状態の意図的継続
